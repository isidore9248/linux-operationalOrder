1、clear或Ctrl+L			清屏
2、init + 级别(0-6)			切换运行级别
3、systemctl set-default  	设置默认运行级别
	+  multi-user.target		三级：多用户有网络服务
     	+  graphical.target		五级：图形界面
4、systemctl get-default		获取当前运行级别
5、man + 指令				获取当前输入指令的帮助文档
6、help  +  指令			功能描述
7、pwd					显示当前文件绝对路径
8、ls   (-a/-l/-al)  			显示当前文件下所有文件
9、cd ~ 					回到家目录，如tom则回						到/home/tom	，root则回到/root
     cd ..					返回上一级目录
     cd /(目录名)				切换到此目录下			
10、mkdir	/(目录名)			创建一级目录
       mkdir -p /(目录名)/(目录名)	创建多级目录			
11、redir	/(目录名)			删除空目录
12、touch (文件名)			创建一个空文件，如path.txt
13、	cp (文件名) (绝对路径)	将(单个)文件复制到该绝对路径下
	cp -r (文件名) (绝对路径)	将该文件夹下所有文件和此目录递归							复制到该绝对路径下
	\cp -r (文件名) (绝对路径)	强制覆盖同名文件复制到该绝对路径

14、	rm -r (文件名)			递归删除目录和目录下文件
	rm -f (文件名)			删除不提示
	rm -rf (文件名)			递归删除目录和目录下文件且不提示
15、	mv  (旧文件名) (新文件名)	重命名
	mv (目录或文件) (路径)	剪切到该路径下
	mv (文件) (路径/文件名)	剪切到该路径下并重命名
16、	cat (文件名)			查看文件内容，不可修改
	cat -n (文件名)			显示行号
	cat -n (文件名) | more	进行交互,
17、	| more	或	more (文件名)		
enter显示下一行，space显示下一页，q立即退出
=输出当前行行号，	:f显示当前文件名和行号	

18、less （文件名）			进行交互，动态加载
space 或  PgDown 向下一页，PgUp 向上一页，
/字符串 或 ？字符串	查找该字符串	n向下查找N向上查找
q	离开

19、	echo	(内容)			将该内容输出
	echo “字符串”		输出 字符串
20、	head (文件名)			查看该文件的前十行，默认为十行
	head -n number (文件名)	查看该文件的前number行
21、	tail (文件名)			查看该文件的末尾十行，默认为十行
	tail -n number	(文件名)	查看该文件的后number行
	tail -f (文件名)			实时追踪文档的更新
22、	>					输出重定向
	ls -l > (文件名)			将列表内容覆盖写入文件中
	cat (文件名1) > (文件名2)	将文件1的内容覆盖到文件2中
23、	>>					追加
	ls -al >> (文件名)		将列表内容追加到文件内
	echo "字符串" >> (文件名)	将字符串追加到文件内
24、	ln -s (原来文件或目录) (软连接名)	创建一个软连接连接到该文件或目						录下，类似快捷方式，删除时用rm
25、	history 				查看所有历史执行命令
	history number			查看最近number条指令
	！8					执行历史编号为8的指令
